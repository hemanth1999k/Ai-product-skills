# /sprint

Full sprint lifecycle support.

## Usage
`/sprint plan [goals and team capacity]`
`/sprint retro [paste sprint notes or describe what happened]`
`/sprint pre-mortem [upcoming sprint or launch]`

## What This Command Does
Plan mode: Uses `sprint-planning`
Retro mode: Uses `retrospective`
Pre-mortem mode: Uses `pre-mortem`
