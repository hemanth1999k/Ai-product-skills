# /write-stories

Transform feature ideas into sprint-ready backlog items.

## Usage
`/write-stories [feature description]`
`/write-stories job [feature for job story format]`

## What This Command Does
Default: Uses `user-story-creation`
Job mode: Uses `job-story-creation`

Outputs: Stories with acceptance criteria in Given/When/Then format.
