# /design-experiment

Design an A/B or multivariate test for a specific hypothesis.

## Usage
`/design-experiment [what you want to test]`
`/design-experiment multivariate [testing multiple elements]`

## What This Command Does
Uses: `ab-test-design` (or `multivariate-test-design`) → full experiment design with sample size, duration, metrics, and decision criteria.
