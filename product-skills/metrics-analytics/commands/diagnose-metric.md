# /diagnose-metric

Systematic root cause analysis for any metric change.

## Usage
`/diagnose-metric [metric name] dropped [X%] starting [when]`

## What This Command Does
Uses: `rca-tropic` → full TROPIC analysis with hypothesis table, root cause candidates, and investigation steps.
