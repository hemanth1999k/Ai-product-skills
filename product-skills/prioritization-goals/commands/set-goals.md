# /set-goals

Define a coherent goal hierarchy from vision to quarterly OKRs.

## Usage
`/set-goals [team context and current priorities]`

## What This Command Does
Uses: `okr-creation` (from product-execution) + `short-long-term-goals`

Outputs: OKR set + Now/Next/Later roadmap framing with horizon analysis.
