# /qa-plan

Generate a comprehensive QA test plan.

## Usage
`/qa-plan [feature or release description]`

## What This Command Does
Uses: `qa-test-plan` + `test-scenario-generation` + `acceptance-criteria`

Outputs: Full test plan with test cases, priority classification, and exit criteria.
