# /validate-idea

Rapid validation framework for any product idea.

## Usage
`/validate-idea [idea description]`

## What This Command Does
Chains: `assumption-mapping` → `lean-validation-framework` → `concept-testing`

Outputs: Hypothesis, MVP type, success criteria, and validation timeline.
