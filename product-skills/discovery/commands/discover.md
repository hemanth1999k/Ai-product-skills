# /discover

Runs a full product discovery cycle for a new idea or problem.

## Usage
`/discover [product idea or problem statement]`

## What This Command Does
Chains 4 skills in sequence:

1. **brainstorm-product-ideas** — Generate diverse solution directions
2. **first-principles-thinking** — Challenge assumptions, find fundamental truths
3. **assumption-mapping** — Map and prioritize the riskiest assumptions
4. **concept-testing** — Design the right validation approach

## Output
By end of this command, you'll have:
- A set of product directions with different angles
- Key assumptions mapped and prioritized
- A validation plan for the top 1-2 ideas

## Example
`/discover A tool that helps solo founders manage customer relationships without a CRM`
