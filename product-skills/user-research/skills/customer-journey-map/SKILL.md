# Customer Journey Map

## Purpose
Visualize the complete customer experience to identify moments of truth, friction points, and opportunity areas.

## Journey Map Framework

### Axis 1: Journey Stages
Define the stages the user moves through:
**Generic stages**: Awareness → Consideration → Decision → Onboarding → Value Realization → Retention → Advocacy

**Product-specific example (SaaS)**:
Aware → Sign Up → Activate → Core Loop → Expand → Renew → Refer

Customize stages to match your actual product lifecycle.

### Axis 2: Swimlanes (for each stage, capture all of these)

**1. User Actions**
What is the user actually doing at this stage?
(Specific behaviors, not "browsing")

**2. Thoughts**
What is the user thinking?
"Is this legit?" / "How long will this take?" / "I don't understand X"

**3. Emotions**
How does the user feel? (Use a sentiment curve — map highs and lows)
😤 Frustrated → 😐 Neutral → 😊 Satisfied → 😍 Delighted

**4. Touchpoints**
Every interaction with your brand at this stage:
- Digital: App, email, website, ads, notifications
- Human: Sales, support, onboarding call
- Indirect: Reviews, word of mouth

**5. Pain Points**
Where does the experience break down?
Quantify if possible: "47% drop off at step 3 of onboarding"

**6. Opportunities**
What improvement could change this stage?
Rate each opportunity: Impact (H/M/L) × Effort (H/M/L)

### Moments of Truth
Mark 2-3 critical moments that most determine whether the user succeeds or fails:
- First moment: First impression / first use
- Critical moment: The moment they realize (or fail to realize) value
- Final moment: Renewal / churn decision point

## Journey Map Process

1. **Align on scope**: Which persona? Which journey? Start-to-end or specific segment?
2. **Gather data**: Interviews, analytics, support data, session recordings
3. **Draft stages**: Work from real user behavior, not internal process
4. **Fill swimlanes**: Data-driven, not assumed
5. **Identify opportunities**: Score and prioritize
6. **Validate**: Show users and ask "does this reflect your experience?"

## Output Format
- Journey map table (stages × swimlanes)
- Sentiment curve visualization
- Top 5 priority opportunity areas ranked by impact × ease
- "Moment of truth" highlights
