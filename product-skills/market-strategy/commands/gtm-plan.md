# /gtm-plan

Build a complete go-to-market strategy.

## Usage
`/gtm-plan [product description and target market]`

## What This Command Does
Chains: `beachhead-segment` → `ideal-customer-profile` → `value-proposition` → `gtm-strategy`

Outputs: Beachhead selection, ICP definition, VP, and full GTM strategy.
